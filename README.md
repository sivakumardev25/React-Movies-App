## Description

A React-based movie search application that integrates with the OMDB API. Users can search for movies, filter by type, view detailed information, and browse results with pagination.

## Features

- Search movies using OMDB API
- Filter by Movie / Series using API (no array.filter used)
- Pagination support
- Detailed movie view (Poster, Title, Year, Genre, Plot, Actors, Rating)
- Error handling and No-results UI
- Responsive modern UI with Tailwind CSS
- React Router navigation

## Tech Stack

- ReactJS
- React Router
- JavaScript
- Tailwind CSS
- OMDB API

## API Used

API: https://www.omdbapi.com/
